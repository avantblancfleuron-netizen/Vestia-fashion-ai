# Vestia

Vestia is an AI-powered fashion intelligence platform designed to transform how people discover, style, and shop fashion.

## Core Features
- AI Style Profiling
- Smart Outfit Generation
- Wardrobe Intelligence
- Fashion Commerce Integration
- Visual Recommendation Engine

## Tech Stack
- React Native
- Supabase
- FastAPI
- OpenAI
- Stripe

## Mission
To build the future operating system for personal style.

## Status
Currently in active development.
